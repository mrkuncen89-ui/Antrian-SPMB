⚠ Reset Semua Antrian Hari Ini

Tindakan ini akan menghapus seluruh data antrian hari ini dan mereset semua loket ke kondisi awal. Backup database akan dibuat secara otomatis sebelum reset dilakukan. Tindakan ini tidak dapat dibatalkan.

Reset Otomatis Tengah Malam

Sistem secara otomatis mereset semua antrian setiap tengah malam (00:00). Backup otomatis dibuat sebelum setiap reset. Sistem memeriksa jadwal reset setiap 60 detik.
